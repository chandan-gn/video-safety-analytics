This is used to finetune Yolo model using SH17-Dataset which has Personal Protective Equipment images.

The original repo is https://github.com/ahmadmughees/SH17dataset/releases
The dataset is in kaggle https://www.kaggle.com/datasets/mugheesahmad/sh17-dataset-for-ppe-detection

Using the above dataset to finetune YOLO Model "yolo8m.pt".
